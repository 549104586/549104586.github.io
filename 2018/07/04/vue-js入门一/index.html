前端框架入门一

2018.07.04 chen  热度 ℃

接手微信公众号已经有两三个月了，但是对于前端架构还了解甚少，一是没有深入学习，二是没有及时总结,导致技术提升非常慢，每天都是新的知识点，三是学习的方法存在问题。现在就开始慢慢撸，构建自己的知识体系。

一 调整前端学习路线

针对第三点做出调整：所谓知识体系，就是知识与知识连接的网络图，而知识框架，就是有层次的树形结构。

在前端的知识框架中需要依次了解的有：

  1. 运行环境

  2. 编码开发

  3. 库类

  4. 理论知识

二 了解前端基础框架

1.这次项目的前端基础框架: vue+webpack+WEUI

首先了解下他们是什么，看看官方背书~

“
vue.js是2018年流行的js框架,号称“全家桶”，是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。

Webpack是一个自动化打包工具, 通过一个入口文件, 将所有依赖的模块打包成为一个js文件, 当然你也可以通过code spliting将其切分为多个js文件, 按需加载以提高效率.

WeUI是一套同微信原生视觉体验一致的基础样式库，由微信官方设计团队为微信 Web 开发量身设计，可以令用户的使用感知更加统一
“

个人感觉vue组件化理念确实很好，webpack之前没接触过，自动化工具只有耳闻，WEUI由于目前是很低版本的，感觉不太符合我的审美。本次项目是开发微信公众号,[ 个人项目开发微信小程序]。

更多的需要整理更新。
